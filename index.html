Acerca de

El GESTOR DE TAREAS es una aplicación web creada como proyecto final en el marco del programa Programando Paraguay.

El gestor de tareas se enfoca en la creación, modificación y eliminación de tareas, permitiendo al usuario asignar diferentes niveles de prioridad y marcarlas como completadas.

La aplicación web ha sido desarrollada utilizando herramientas como HTML, CSS y JavaScript. Con HTML se ha estructurado la página, CSS se ha utilizado para estilizarla y JavaScript se ha utilizado para hacerla dinámica mediante la manipulación del DOM.
